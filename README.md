# Mocker - Entry Test Master 🚀

**The ultimate platform for entry test preparation**

Master engineering & medical entrance exams with free, comprehensive mock tests and detailed analytics.

## Features

✨ **1000+ Practice Questions** - Comprehensive question bank covering all entry test subjects

⚡ **Full-Length Mock Tests** - Experience realistic test conditions with timed examinations

📊 **Detailed Analytics** - Track progress with in-depth performance reports

🎯 **Smart Learning Path** - Personalized recommendations based on your performance

👥 **Community Support** - Connect with thousands of test-takers

📱 **Anytime, Anywhere** - Practice on desktop, tablet, or mobile devices

🎓 **100% Free** - No hidden costs, no premium features

## Tech Stack

- **Framework**: Next.js 16 with React 19
- **Styling**: Tailwind CSS with custom black/white theme
- **UI Components**: Radix UI components
- **Forms**: React Hook Form
- **Icons**: Lucide React
- **Analytics**: Vercel Analytics
- **Themes**: Next Themes for dark mode support

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or pnpm

### Installation

1. Clone the repository:
```bash
cd clone-nustrive-website
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment to Vercel

### Option 1: Using Vercel CLI

```bash
npm install -g vercel
vercel
```

### Option 2: Using GitHub Integration

1. Push to GitHub:
```bash
git init
git add .
git commit -m "Initial Mocker release"
git branch -M main
git remote add origin https://github.com/yourusername/mocker.git
git push -u origin main
```

2. Connect to Vercel:
   - Go to [vercel.com](https://vercel.com)
   - Sign in with GitHub
   - Click "New Project"
   - Select your repository
   - Click "Deploy"

### Option 3: Direct Deploy Button

Click the button below to deploy directly to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/mocker)

## Color Scheme

The application uses a beautiful **black and white theme**:

- **Light Mode**:
  - Background: #ffffff (White)
  - Foreground: #000000 (Black)
  - Primary: #000000 (Black)
  
- **Dark Mode**:
  - Background: #0a0a0a (Very Dark)
  - Foreground: #ffffff (White)
  - Primary: #ffffff (White)
